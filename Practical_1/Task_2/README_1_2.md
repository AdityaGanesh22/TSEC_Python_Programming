# Assignment 2: Calculating Areas of Geometric Figures

## Objective
Write three Python functions to calculate the area of common geometric figures.

## Instructions
- Implement the following functions in `student_code_1_2.py`:
  1. `area_circle(radius)`
     - Formula: pi * radius^2
  2. `area_rectangle(length, width)`
     - Formula: length * width
  3. `area_triangle(base, height)`
     - Formula: 0.5 * base * height

- Each function should return the area as a float.

- Examples:
```python
area_circle(3)
# Output: 28.274333882308138

area_rectangle(4, 5)
# Output: 20

area_triangle(6, 8)
# Output: 24
```
