# Assignment 5: Calculating Simple Interest

## Objective
Write a Python function to calculate the simple interest based on user input.

## Instructions
- Implement the following function in `student_code_1_5.py`:
  1. `calculate_simple_interest(principal, rate, time)`
     - Formula: SI = (Principal × Rate × Time) / 100

- The function should return the simple interest as a float.

- Examples:
```python
calculate_simple_interest(1000, 5, 2)
# Output: 100.0

calculate_simple_interest(5000, 10, 1)
# Output: 500.0
```
