# Assignment 6: Exploring Basic Arithmetic Operations in Python

## Objective
Write Python functions to explore basic arithmetic operations.

## Instructions
- Implement the following functions in `student_code_1_6.py`:
  1. `add_numbers(a, b)` → returns a + b
  2. `subtract_numbers(a, b)` → returns a - b
  3. `multiply_numbers(a, b)` → returns a * b
  4. `divide_numbers(a, b)` → returns a / b
  5. `modulus_numbers(a, b)` → returns a % b

- Each function should return the result of the operation.

- Examples:
```python
add_numbers(10, 5)
# Output: 15

subtract_numbers(10, 5)
# Output: 5

multiply_numbers(10, 5)
# Output: 50

divide_numbers(10, 5)
# Output: 2.0

modulus_numbers(10, 5)
# Output: 0
```
