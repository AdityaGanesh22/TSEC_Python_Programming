# Assignment 3: Developing Conversion Utilities

## Objective
Write three Python functions to perform basic conversions.

## Instructions
- Implement the following functions in `student_code_1_3.py`:
  1. `rupees_to_dollars(rupees)`
     - Assume 1 USD = 80 INR
     - Formula: rupees / 80
  2. `celsius_to_fahrenheit(celsius)`
     - Formula: (celsius * 9/5) + 32
  3. `inches_to_feet(inches)`
     - Formula: inches / 12

- Each function should return the converted value as a float.

- Examples:
```python
rupees_to_dollars(160)
# Output: 2.0

celsius_to_fahrenheit(0)
# Output: 32.0

inches_to_feet(24)
# Output: 2.0
```
